# Healthcare AI Prediction System – Web Application

The Healthcare AI Prediction System is a complete web application built using modern web technologies. The application is structured to support healthcare-related workflows such as triage, analytics, staff management, and resource viewing through a clean and scalable website architecture.

## Project Overview
This project is a fully structured healthcare web platform developed with a modern component-based architecture. It focuses on delivering a professional user experience while being ready for integration with backend APIs and AI-powered services.

The application demonstrates strong skills in web application development, modular UI design, routing, and scalable project organization.


## Core Features
- Multi-page healthcare web application using modern routing  
- Dashboard-style layout for structured data presentation  
- Dedicated sections for healthcare operations and analytics  
- Clean UI components and reusable layouts  
- API-ready architecture for dynamic data integration  

## Functional Modules
Based strictly on the existing application structure:

## 1. Healthcare Triage Module
- Dedicated triage page for healthcare-related assessment workflows  
- Designed to support patient prioritization or AI-assisted decision flows  

## 2. Analytics Module
- Analytics section structured for healthcare data insights  
- Includes chart-based components for data visualization  
- Designed for future integration with real-time or AI-generated analytics  

## 3. Staff Management Module
- Separate staff section to manage or display healthcare staff-related information  
- Modular design suitable for role-based expansion  

## 4. Resources Module
- Resource page to display healthcare-related resources or information  
- Structured for scalability and content expansion  

## 5. API Integration Layer
- API route structure included for backend connectivity  
- Fetch utility implemented for handling API communication  

## UI & Architecture Features
- Component-based UI system  
- Reusable layout and header components  
- Theme provider for consistent styling  
- Responsive design support  
- Utility hooks for mobile handling and notifications  


## Technology Stack
- **Next.js** – Full web application framework  
- **React** – Component-based UI development  
- **TypeScript** – Type-safe development  
- **Tailwind CSS** – Modern and responsive styling  
- **PostCSS** – CSS processing  
- **PNPM** – Dependency management  

## Business Value
This project represents a production-ready healthcare web platform foundation. It is designed to support healthcare workflows, data visualization, and AI-assisted systems while maintaining clean code organization and scalability.


## How to Run the Project Locally

## Prerequisites
- Node.js (v18 or above recommended)
- PNPM or NPM installed

## Steps to Run

## Clone the repository
git clone <your-repository-url>

## Navigate to the project directory
cd healthcare-ai

## Install dependencies
pnpm install
 or
npm install

## Start the development server
pnpm dev
or
npm run dev
